C# Windows Forms ile geliştirdiğim bu uygulama, işletmelerin personel izin süreçlerini kolayca yönetebilmesini sağlar. Kullanıcı dostu ve sade bir arayüze sahiptir. Uygulamada kullanıcı ve admin girişleri ayrı tutulmuştur.
Kullanıcılar izin taleplerini oluşturabilirken, admin paneli üzerinden izin talepleri görüntülenip onay veya ret işlemleri yapılabilmektedir.
